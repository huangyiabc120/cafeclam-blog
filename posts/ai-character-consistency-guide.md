# 角色一致性指南：防换脸与质感升级

> AI漫剧最大的坑就是"换脸"——同一集里角色长相变了。本文系统讲解如何用三视图+Lora等技术锁定角色。

---

## 一、换脸问题深度解析

### 为什么AI会换脸？

| 原因 | 解释 |
|------|------|
| 提示词不够具体 | 没有锁定足够的角色特征 |
| 参考图不够清晰 | 缺少明确的视觉锚点 |
| 镜头角度变化大 | 侧脸/背面时特征丢失 |
| 光影风格变化 | 不同光影下角色辨识度下降 |
| 模型特性 | 某些模型对人脸理解有偏差 |

### 换脸的心理学解释

AI生成的是"概念"而非"个体"。如果提示词中的特征不够强，AI会生成它认为"更匹配"的另一个类似角色。

**解决方案的核心思路：提供足够多的视觉信息，让AI锁定这个"个体"。**

---

## 二、三视图防换脸系统（核心）

### 为什么三视图有效？

三视图从三个正交角度提供角色的完整视觉信息，让AI建立对角色的立体认知。

```
正面 → 知道五官结构
侧面 → 知道轮廓/鼻梁/下颌
背面 → 知道发型/肩线/背影
```

### 三视图标准模板

#### 正面照（Front View）

```
正面照，精致五官，
[脸型：椭圆/方圆/瓜子/长方]，
[眉形：剑眉/平眉/粗眉/细眉]，
[眼型：双眼皮/丹凤眼/下垂眼/圆眼]，
[鼻型：高鼻/直鼻/翘鼻/蒜头鼻]，
[唇形：薄唇/厚唇/微笑唇]，
[肤色：白皙/古铜/小麦色/黄调]，
[发型：长度+颜色+造型]，
清晰视线，镜头正对，无角度偏斜，
正面光源，均匀照明，五官细节清晰
```

#### 侧面照（Side Profile）

```
标准90度侧脸，
[侧脸轮廓：颧骨突出/柔和/凹陷]，
[下颌线：方形/圆形/V形]，
[鼻梁：直/鹰钩/塌鼻]，
[耳廓：贴面/招风耳]，
[发型侧面：发尾走向/耳边碎发]，
侧向光源，轮廓光强化，
阴影过渡自然，立体感强
```

#### 背面照（Back View）

```
背面全身照，镜头位于正后方，
[发型背面：发尾造型/分缝/后颈线条]，
[肩部轮廓：宽肩/窄肩/溜肩]，
[服装背面：领口/口袋/扣子/腰带]，
背部中心线对齐，无偏移，
背光光源，轮廓逆光，
统一背景，与正面取景一致
```

### 三视图使用流程

```
Step 1: 生成三视图 → 选取最满意的一组
Step 2: 三张图同时作为后续参考
Step 3: 后续所有镜头都以这三张为锚点
Step 4: 如需换装/换场景 → 仅改变服装/背景，保持脸部参考
```

## 三、三视图防换脸提示词公式

### 角色锁定公式

```
【固定不变】（脸/眉/眼/发型/肤色）
[详细描述，全部要写清楚]

【可变元素】（服装/场景/道具/光影）
[可以改变的描述]

【风格锚点】
masterpiece, best quality, cinematic lighting
```

### 具体示例

**正确示范：**
```
【文生图提示词】
角色参考图：[上传三视图]
Prompt：
[女性角色，瓜子脸，眉形英气，
眼型为丹凤眼，鼻梁高挺，
肤色白皙，长发及腰黑色，
挽起发髻，古装白色长裙]
站在[竹林]中，[持剑]姿态，
[侧光，剑气效果]，
[武侠古风风格]
--style raw
--cref [上传三视图]
--cw 80
```

**错误示范：**
```
"美丽的女人在竹林中持剑"
（没有锁定任何特征，AI自由发挥，必然换脸）
```

## 四、角色一致性工作流（实战）

### 完整工作流程

```
【Phase 1: 建立阶段】
1. 编写详细角色描述（≥200字）
2. 生成三视图 → 选择最佳
3. 三视图作为角色资产保存

【Phase 2: 应用阶段】
1. 后续所有镜头 → 上传三视图作为参考
2. 提示词结构：[场景描述] + [动作/表情] + [服装/道具] + [风格锚点]
3. 不在提示词中重复描述脸部特征

【Phase 3: 修正阶段】
1. 如出现换脸 → 重新上传更清晰的参考图
2. 使用Inpaint局部重绘脸部
3. 使用ControlNet锁定角色特征

【Phase 4: 稳定阶段】
1. 确认不换脸后 → 可尝试训练Lora进一步固定
2. 定期用相同参考测试一致性
```

## 五、SSS皮肤质感详解

### 什么是SSS？

SSS（Subsurface Scattering）= 次表面散射，指光线穿透半透明材质（如皮肤、蜡烛、树叶）后散射的效应。

### SSS皮肤效果

| 无SSS | 有SSS |
|-------|-------|
| 塑料感/蜡像感 | 真实皮肤/通透感 |
| 光滑但假 | 有细节但柔润 |

### SSS皮肤提示词

```
subsurface scattering skin,
translucent skin edges,
light penetrating dermis,
natural skin glow,
warm skin undertone,
soft light diffusion,
realistic human skin shader
```

### 不同风格的SSS强度

| 风格 | SSS强度 | 说明 |
|------|---------|------|
| 写实摄影 | 高 | 阳光/柔光箱+反光板 |
| 古风/武侠 | 中 | 侧光+轮廓光 |
| 赛博朋克 | 低 | 强对比+霓虹 |
| 动漫/二次元 | 无 | 硬边光+平面感 |

## 六、BJD风格应用

### BJD（Ball Joint Doll）特点

球关节娃娃风格：精致、均匀肤色、统一比例、哑光质感。

### BJD vs 写实对比

| 特征 | BJD | 写实摄影 |
|------|-----|---------|
| 皮肤质感 | 哑光/瓷质 | 自然毛孔 |
| 眼睛 | 玻璃感/光泽 | 虹膜纹理 |
| 比例 | 风格化/均匀 | 真实比例 |
| 关节 | 可见接缝 | 不可见 |

### BJD风格提示词

```
ball jointed doll style,
porcelain skin, matte finish,
delicate facial features,
large glass-like eyes,
seamless joints,
matte makeup, doll-like,
high detail, clean design
```

## 七、光影三件套

### 光影三件套 = 主光 + 补光 + 轮廓光

#### 1. 主光（Key Light）

**位置：** 侧面45度

**作用：** 提供主要照明和阴影

**提示词：**
```
key light, 45-degree angle,
main illumination,
hard or soft quality,
warm or cool tone
```

#### 2. 补光（Fill Light）

**位置：** 主光对侧，较弱

**作用：** 填充阴影，保持细节

**提示词：**
```
fill light, opposite side,
reduced intensity,
soft shadows,
balanced exposure
```

#### 3. 轮廓光（Rim Light）

**位置：** 背后/侧后方

**作用：** 勾勒轮廓，与背景分离

**提示词：**
```
rim light, backlight,
edge lighting, hair light,
separation from background,
color rim glow
```

### 三件套速查表

| 风格 | 主光 | 补光 | 轮廓光 |
|------|------|------|--------|
| 写实 | 柔光箱45° | 0.5强度 | 白色/金色 |
| 古风 | 侧光 | 极弱 | 暖色 |
| 赛博 | 硬光单侧 | 无 | RGB霓虹 |
| 悬疑 | 底光 | 无 | 绿色 |
| 浪漫 | 蝴蝶光 | 侧补 | 暖橙 |

## 八、角色一致性检查清单

### 每次生成前检查

- [ ] 三视图是否清晰？
- [ ] 参考图是否上传？
- [ ] 提示词是否锁定了脸部特征？
- [ ] 光影风格是否与角色设定一致？

### 每次生成后检查

- [ ] 脸部是否与参考一致？
- [ ] 肤色是否统一？
- [ ] 眼睛形状是否正确？
- [ ] 发型是否保持？

### 出现换脸时的解决步骤

```
Step 1: 确认换脸程度（轻微/严重）
Step 2: 轻微 → 调整参考强度(cw值)
Step 3: 严重 → 重新生成参考图
Step 4: 使用Inpaint局部重绘
Step 5: 考虑训练角色Lora
```

## 九、不同场景的换装处理

### 核心原则

```
换装不变脸：只描述服装变化，不描述脸部
```

### 示例

**日常场景：**
```
Prompt：
[角色三视图参考]
[角色]穿[休闲服装描述]，
站在[咖啡厅]，[喝咖啡]动作，
[柔光，温馨氛围]
```

**战斗场景：**
```
Prompt：
[角色三视图参考]
[角色]穿[战斗服装描述]，
站在[战场]，[持剑]姿态，
[侧光，战斗氛围]
```

**注意：** 服装描述要详细，但脸部特征不要重复描述（让参考图来处理）。

## 十、4月备案新规注意事项

根据2026年4月起的备案新规：

### 内容合规

- 禁止出现真实人脸的高度相似克隆
- 禁止未经授权的知名IP角色生成
- 禁止过度暴露或不当内容

### 技术建议

- 使用原创角色设计
- 避免与真实明星脸型高度相似
- 建议使用艺术化/风格化处理

### 备案过审技巧

- 角色设计具有足够的原创性
- 避免直接复制已知角色的标志性特征
- 服装/造型具有辨识度

## 结语

角色一致性是AI漫剧的生命线。没有一致的角色，就没有真正的叙事。

记住三个核心：
1. **三视图锁定** — 从三个角度建立完整的角色视觉
2. **参考图优先** — 提示词说不清楚的，让参考图来
3. **换装不变脸** — 只改变服装/场景，不改变脸部描述
