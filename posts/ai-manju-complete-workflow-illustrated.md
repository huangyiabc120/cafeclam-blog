## ① 角色设计：三视图定位

### 三视图是什么？

三视图是角色的"基因档案"，从正面、侧面、背面三个角度锁定角色长相，让AI在所有镜头里保持角色一致。

```
        正面（Front）           侧面（Side）           背面（Back）
        
           ┌─┐                ┌─┐                 ┌─┐
          │◯│                │ │                 │◯│
          │ │    ←正面→     │─│   ←侧面→       │ │    ←背面→
           └─┘                 └─┘                  └─┘
         五官结构             轮廓/鼻梁             发型/肩线
```

### 三视图标准模板

**正面照：**
```
脸型（椭圆/方圆/瓜子）+
眉形（剑眉/平眉）+
眼型（双眼皮/丹凤眼）+
鼻型（高鼻/直鼻）+
唇形（薄唇/厚唇）+
肤色（白皙/古铜/小麦）+
发型（长度+颜色+造型）
```

**侧面照：**
```
标准90度侧脸+
颧骨轮廓（突出/柔和）+
下颌线（方形/V形）+
鼻梁（直/鹰钩）+
发型侧面（发尾走向）
```

**背面照：**
```
发型背面（发尾造型）+
肩部轮廓（宽肩/窄肩）+
服装背面（领口/腰带）
```

## ② 风格确定：选择你的视觉语言

### 八大风格速查

| 风格 | 运镜 | 灯光 | 构图 | 代表作品 |
|------|------|------|------|---------|
| **武侠/动作** | 动态+快速 | 侧光剑气 | 纵深三分 | 《卧虎藏龙》 |
| **情感/爱情** | 缓慢推进 | 柔光逆光 | 框架亲密 | 《花束般的恋爱》 |
| **悬疑/惊悚** | 固定+慢移 | 顶光低对比 | 低角压迫 | 《沉默的羔羊》 |
| **古风/仙侠** | 诗意摇移 | 逆光留白 | 对称写意 | 《英雄》《聂隐娘》 |
| **赛博/科技** | 俯仰倾斜 | 霓虹RGB | 荷兰角 | 《银翼杀手2049》 |
| **宣传片** | 稳定跟拍 | 写实自然 | 三分构图 | Apple广告 |
| **短视频** | 快切+特写 | 强对比 | 竖屏适配 | 抖音爆款 |
| **文艺** | 慢镜头 | 弗莱塔格 | 长镜头 | 艺术电影 |

## ④ 灯光设计：光影是情绪的语言

### 灯光三件套

```
                    主光 (Key Light)
                   侧面45度角
                        │
                        ▼
    轮廓光 ◄── 主体 ──► 补光
    (Rim Light)   (Subject)   (Fill Light)
    背后冷色      ←────────────────→  对侧弱
```

## ⑤ 运镜技巧：镜头是导演的语言

### 基础运镜六式

```
   ① 推进 (Push In)     ② 拉远 (Pull Out)
        ┌───┐               ┌───┐
        │ → │→              │ → │→
        └───┘               └───┘
        聚焦强调              揭示全貌

   ③ 横移 (Pan)          ④ 竖移 (Tilt)
        ┌───┐               ┌───┐
        │→→→│               │ ↑ │
        │   │               │ ↑ │
        │   │               │ ↓ │
        └───┘               └───┘
        扫描环境             仰望/俯视

   ⑤ 跟拍 (Follow)       ⑥ 环绕 (Orbit)
        ┌───┐               ┌───┐
        │ ● │→              │ ● │
        │   │                ╲  │
        └───┘                 ╲ │
                              ╲│
        跟随主体               旋转
```

### 运镜与情绪对应图

```
─────────────────────────────────────

浪漫       缓慢推进+环绕       快切

动作       跟拍+环绕+快切     全固定
```

## ⑦ AI生成：把分镜变成画面

### 提示词结构公式

```
【文生图提示词结构】

主体 + 动作 + 场景 + 景别 + 运镜 + 光影 + 构图 + 风格 + 质量锚点

示例：
```

## ⑧ 剪辑合成：从图片到视频

### 剪辑节奏公式

```
开场(15%)：全景/中景建立 ── 2~3个镜头 ── 慢节奏
  │
发展(50%)：中景/近景交替 ── 快慢结合 ── 逐步加速
  │
高潮(25%)：特写/大特写 ── 快切/定格 ── 情绪爆发
  │
结尾(10%)：拉远/黑场 ── 截断 ── 期待下一集
```

### 卡一半剪辑法图解

```
❌ 错误展示（完整展示，无悬念）：

镜头1：两人对视
  ↓
镜头2：拔剑出鞘
  ↓
镜头3：剑身碰撞
  ↓
镜头4：一方倒地
  ↓
镜头5：胜者站立
→ 完整流程，毫无悬念

✅ 正确做法（卡一半）：

镜头1：两人对视
  ↓
镜头2：拔剑出鞘
  ↓
镜头3：剑身即将碰撞 ⚡
  ↓
  黑场截断！
→ 观众：然后呢然后呢？？
```
