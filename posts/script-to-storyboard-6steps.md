# 从剧本到分镜的6步法：我的实战经验总结

> 很多人在做漫剧时拿到剧本就直接开始生成，结果发现镜头混乱、风格不统一、节奏失控。我总结了一套"从剧本到分镜"的6步法，帮我从B级做到了S级。

---

## 一、我曾经的混乱经历

刚开始做漫剧时，我是这样工作的：

```
拿到剧本 → 直接想镜头 → 边生成边改 → 最后剪辑
```

结果：
- 镜头和剧本对不上
- 风格飘忽不定
- 时长完全失控
- 做完才发现节奏有问题

**教训：漫剧是"设计"出来的，不是"生成"出来的。**

---

## 二、6步法的核心逻辑

```
剧本 → 情感曲线 → 风格确定 → 镜头规划 → 运镜设计 → 分镜输出
```

**核心思想：先想清楚再动手。**

---

## 三、6步详解

### Step 1：剧本结构分析

**目标：** 理解剧本的"骨架"

**怎么做：**

1. **通读剧本**（不要边读边分析，先完整看一遍）
2. **标记三个关键点：**
   - **Hook点**：开头用什么抓住观众？
   - **转折点**：情绪/情节在哪里转折？
   - **截断点**：在哪里切断，让观众期待下一集？

3. **画出情感曲线：**

```
情感曲线示例（武侠对峙）：
    高
   /  \
  /    \_______
 /            \
/              \____
低
0秒  15秒  30秒  45秒  60秒
```

**输出模板：**
```
【项目】
【时长】

【Hook点】
用什么在3秒内抓住观众？

【情感转折点】
- 第一次转折：
- 第二次转折：
- 高潮点：

【截断点】
在哪里切断，让观众追更？
```

---

### Step 2：景别分配规划

**目标：** 决定"用什么大小的画面讲故事"

**核心原则：**
- 景别决定情绪距离
- 远景=观众是旁观者
- 特写=观众是当事人

**景别与情绪对应：**

| 景别 | 情绪效果 | 适用时机 |
|------|---------|---------|
| 大远景 | 渺小感、命运感 | 开场建立、结尾 |
| 全景 | 完整场景、关系 | 动作全景、关系展示 |
| 中景 | 叙事感、日常 | 对话、行动 |
| 近景 | 情绪参与 | 情感表达 |
| 特写 | 强烈情绪 | 高潮、关键瞬间 |
| 大特写 | 极度聚焦 | 极致情绪、物品强调 |

**景别分配公式：**

```
开场（前15%）：远景/全景 → 建立世界
发展（中50%）：中景/近景 → 叙事参与
高潮（后25%）：近景/特写 → 情绪爆发
结尾（10%）：拉远/留白 → 截断期待
```

**经验：不要连续使用相同的景别。** 比如连续3个特写会让人窒息，连续3个全景会让人失去焦点。

---

### Step 3：运镜方式选择

**目标：** 决定"镜头怎么动"

**运镜与情绪对应：**

| 运镜 | 情绪效果 | 禁忌 |
|------|---------|------|
| 固定 | 冷静、观察、沉默 | 动作场景不要全固定 |
| 推进 | 聚焦、强调、压迫 | 不要在放松场景用 |
| 拉远 | 揭示、放松、释然 | 不要在紧张时用 |
| 横移 | 扫描、展示、过渡 | 不要在情绪高点用 |
| 环绕 | 沉浸、卷入、浪漫 | 不要太快 |
| 升降 | 气势、戏剧性 | 开场慎用 |

**运镜节奏设计：**

```
紧张场景：
固定(2秒) → 推进(1秒) → 固定(1秒) → 快切

情感场景：
固定(4秒) → 环绕(3秒) → 拉远(3秒)

动作场景：
推进(0.5秒) → 固定(0.5秒) → 横移(1秒) → 推进(0.5秒)
```

**教训：情绪强烈的场景，运镜要克制。**

---

### Step 4：灯光氛围设计

**目标：** 决定"用什么光影讲故事"

**灯光与情绪对应：**

| 情绪/场景 | 灯光方案 | 关键词 |
|---------|---------|--------|
| 紧张/危险 | 硬光+明暗对比 | side lighting, high contrast |
| 悲伤/失落 | 暗调+冷色 | low key, blue undertone |
| 浪漫/温馨 | 柔光+暖色 | soft lighting, warm tone |
| 悬疑/神秘 | 单一光源+阴影 | single light, deep shadow |
| 动作/打斗 | 侧光+剑光 | side lighting, blade gleam |
| 史诗/宏大 | 逆光+轮廓 | backlight, silhouette |

**教训：同一部剧里，灯光风格要统一。不同场景可以微调，但基调要一致。**

---

### Step 5：构图设计

**目标：** 决定"画面怎么布局"

**构图与情绪对应：**

| 构图 | 情绪效果 | 适用场景 |
|------|---------|---------|
| 三分法 | 稳定、通用 | 所有场景基础 |
| 中心 | 聚焦、强调 | 主角特写 |
| 框架 | 聚焦+封闭感 | 悬疑、审讯 |
| 留白 | 意境、想象 | 古风、文艺 |
| 对角线 | 动感、张力 | 动作、冲突 |
| 荷兰角 | 失衡、紧张 | 悬疑、醉酒 |

**构图与景别配合：**

| 景别 | 推荐构图 |
|------|---------|
| 远景 | 三分法、中心 |
| 全景 | 三分法、对角线 |
| 中景 | 三分法、框架 |
| 近景 | 中心、留白 |
| 特写 | 中心、紧凑 |

---

### Step 6：分镜脚本输出

**目标：** 把前面的分析成果写成可执行的脚本

**标准分镜格式：**

| 序号 | 时长 | 场景 | 景别 | 提示词 | 运镜 | 灯光 | 构图 | 音效 |
|------|------|------|------|---------|------|------|------|------|
| 1 | 5秒 | 竹林 | 远景 | XXXX | 固定 | 自然光 | 三分法 | 风 |
| 2 | 3秒 | 剑客 | 中景 | XXXX | 横移 | 侧光 | 三分法 | 脚步 |
| 3 | 2秒 | 手部 | 特写 | XXXX | 推进 | 顶光 | 中心 | 拔剑 |

**提示词写作公式：**

```
【文生图】
主体 + 动作 + 场景 + 景别 + 运镜 + 光影 + 构图 + 风格 + 质量锚点

示例：
武侠剑客在竹林中拔剑，中景镜头，
横移跟随，侧面光源，剑气寒光效果，
高对比度，三分法构图，
写实摄影风格，电影感色调
```

**分镜完成后的自检：**

- [ ] 时长总和是否达标？
- [ ] 景别是否有变化？（避免连续同景别）
- [ ] 运镜是否太花哨或太单调？
- [ ] 灯光风格是否统一？
- [ ] 是否有Hook镜头（开头3秒）？
- [ ] 结尾是否截断在期待点？

---

## 四、6步法的使用时机

### 新项目开始时

从头到尾走一遍6步，确认每个环节。

### 中途遇到问题时

回到对应步骤检查：
- 剧本有问题？→ 回到 Step 1
- 镜头混乱？→ 回到 Step 2
- 风格飘了？→ 回到 Step 3-4
- 节奏失控？→ 回到 Step 2-5

### 项目复盘时

用6步法回顾每个环节，找出问题点。

---

## 五、常见错误与修正

| 错误 | 原因 | 修正 |
|------|------|------|
| 景别单调 | 没有规划 | 按公式分配景别 |
| 运镜混乱 | 没有确定风格 | 先确定主运镜风格 |
| 节奏拖沓 | 高潮没截断 | 用"卡一半"原则 |
| 风格飘移 | 灯光没统一 | 确定基调后不改 |
| 提示词混乱 | 没有按公式写 | 按提示词公式结构 |

---

## 结语

6步法的核心是"先想后做"。

每个步骤都是前一步的延伸：
- **Step 1** 告诉你"要讲什么故事"
- **Step 2-5** 告诉你"用什么方式讲"
- **Step 6** 把前面的决策变成可执行的脚本

**记住：漫剧是设计出来的，不是生成出来的。**

祝你做出精心设计的S级作品！
