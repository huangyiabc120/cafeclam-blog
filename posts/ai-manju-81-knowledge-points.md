# AI漫剧81个知识点总结：完整知识体系手册

> 本文整理了AI漫剧创作的全部核心知识点，覆盖剧本、运镜、光影、构图、角色、剪辑6大板块。可以作为速查手册使用。

## 一、剧本与叙事（15个知识点）

### 1. 八大剧本风格

| 风格 | 核心特点 | 运镜 | 光影 | 构图 |
|------|---------|------|------|------|
| 武侠/动作 | 力量感 | 动态+快速 | 侧光剑气 | 纵深三分 |
| 情感/爱情 | 代入感 | 缓慢推进 | 柔光逆光 | 框架亲密 |
| 悬疑/惊悚 | 不安感 | 固定+慢移 | 顶光低对比 | 低角压迫 |
| 古风/仙侠 | 意境感 | 诗意摇移 | 逆光留白 | 对称写意 |
| 赛博/科技 | 未来感 | 俯仰倾斜 | 霓虹RGB | 荷兰角 |
| 宣传片 | 商业感 | 稳定跟拍 | 写实自然 | 三分构图 |
| 短视频 | 快节奏 | 快切+特写 | 强对比 | 竖屏适配 |
| 文艺 | 留白感 | 慢镜头 | 弗莱塔格 | 长镜头 |

### 2. 剧本四元素标注

- **景别**：远/全/中/近/特/大特写
- **人物**：谁在做
- **动作**：做什么
- **情绪**：什么感觉

### 3. Hook公式

开头3秒 = 视觉冲击 + 悬念 + 期待

### 4. 情感曲线设计

```
淡入 → 背景铺陈（慢剪辑）→ 冲突建立（中剪辑）→ 高潮（快剪辑）→ 爆发 → 余韵（慢剪辑+留白）
```

### 5. "卡一半"剪辑法

在高潮将至时立刻切断，永远不展示完整结果

### 6. 时长分配公式

```
开场(15%)：全景/中景建立 → 2-3个镜头
发展(50%)：中景/近景交替 → 节奏推进
高潮(25%)：特写/大特写 → 情绪爆发
结尾(10%)：拉远/留白 → 戛然而止
```

### 7. 对白写作原则

- 简洁有力，每句有存在理由
- 符合角色性格
- 服务叙事，不服务解释

### 8. 叙事结构

- 三幕式（建置/对抗/解决）
- 丹·哈蒙故事圈
- 英雄之旅（12阶段）

### 9. 场景转换技巧

- 动作匹配
- 声音先入
- 遮挡切换
- 黑场淡入

### 10. 竖屏 vs 横屏

| 维度 | 竖屏 | 横屏 |
|------|------|------|
| 视野 | 窄 | 宽 |
| 关注点 | 人物 | 环境 |
| 景别 | 近景/特写多 | 全景/中景多 |
| 平台 | 抖音/快手 | B站/YouTube |

### 11. AI提示词结构

```
主体 + 动作 + 场景 + 景别 + 运镜 + 光影 + 构图 + 风格 + 质量锚点
```

### 12. 分镜脚本标准格式

| 序号 | 时长 | 场景 | 景别 | 提示词 | 运镜 | 灯光 | 构图 | 音效 |

### 13. 剧本格式规范

- 竖屏思维
- 四元素标注
- 时长控制（每分钟约100字）

### 14. 情绪标注速查

| 情绪 | 镜头建议 | 光影建议 |
|------|---------|---------|
| 紧张 | 短镜头、快切 | 硬光、高对比 |
| 悲伤 | 长镜头、留白 | 暗调、冷色 |
| 幸福 | 中等时长 | 柔光、暖色 |
| 愤怒 | 快切、推进 | 侧光、对比 |

### 15. 编剧心法

- "Show, don't tell"（展示，不要告诉）
- 情绪物化为镜头
- 用画面讲故事

## 二、运镜技巧（15个知识点）

### 16. 基础景别

远景/全景/中景/近景/特写/大特写

### 17. 镜头运动

推/拉/摇/移/跟拍/环绕/升降

### 18. 高级运镜技巧

- 慢速推进
- 快速拉远
- 抖动镜头
- 旋转镜头
- Dolly zoom

### 19. 分镜标记速查

CU/MS/LS/OTS/POV/ECU

### 20. 运镜与情绪对应

| 运镜 | 情绪效果 | 禁忌 |
|------|---------|------|
| 固定 | 冷静、观察 | 动作场景不要全固定 |
| 推进 | 聚焦、强调 | 放松场景不要用 |
| 拉远 | 揭示、放松 | 紧张时不要用 |
| 环绕 | 沉浸、卷入 | 不要太快 |
| 横移 | 扫描、展示 | 情绪高点不要用 |

### 21. 导演级运镜50例

详见《运镜技巧50例》文章，包含基础/进阶/高级三类运镜完整提示词模板。

### 22. 三镜头循环公式

1. 极广角+低位仰拍（建立）
2. 眼部极特写+瞳孔倒影（代入）
3. 荷兰角+动作模糊（动感）

### 23. 运镜节奏设计

```
紧张：固定(2s) → 推进(1s) → 固定(1s) → 快切
情感：固定(4s) → 环绕(3s) → 拉远(3s)
动作：推进(0.5s) → 固定(0.5s) → 横移(1s)
```

### 24. POV主观镜头用法

- 代入角色视角
- 增强沉浸感
- 制造紧张/恐惧

### 25. 长镜头运用

- 完整呈现动作
- 挑战与实力展示
- 沉浸感强

### 26. 景别跳切效应

- 大跳切：时间跳跃
- 小跳切：时间延伸
- 逆向跳切：超现实

### 27. 轴线原则

- 180度规则：摄影机在轴线一侧
- 越轴解决方法：中性镜头过渡

### 28. 过肩镜头用法

- 对话场景标配
- 建立关系感
- 视线匹配

### 29. 主观+客观交替

- 客观镜头：建立场景
- 主观镜头：代入角色
- 交替使用：张弛有度

### 30. 运动镜头注意事项

- 动要有理由
- 速度要服务情绪
- 结束要有落点

## 三、光影设计（15个知识点）

### 31. 曝光三要素

| 要素 | 作用 | AI关键词 |
|------|------|---------|
| 光圈 | 进光量+景深 | shallow depth of field |
| 快门 | 运动模糊 | motion blur |
| ISO | 亮度（慎用） | clean image |

### 32. 光质分类

| 类型 | 效果 | 适用 |
|------|------|------|
| 硬光 | 清晰阴影、高对比 | 武侠、戏剧 |
| 柔光 | 柔和过渡、梦幻 | 爱情、唯美 |

### 33. 20种灯光效果

详见《AI视频灯光与运镜指南》文章：蝴蝶光/伦勃朗光/环形光/侧光/逆光/顶光/底光/柔光/硬光/霓虹光/蜡烛光/夕阳光/月光/舞台光/追光/散射光/RGB光/单光源/三点布光

### 34. 灯光与情绪

| 情绪 | 灯光方案 |
|------|---------|
| 紧张 | 硬光+明暗对比 |
| 悲伤 | 暗调+冷色 |
| 浪漫 | 柔光+暖色 |
| 悬疑 | 单一光源+阴影 |
| 动作 | 侧光+剑光 |

### 35. 三点布光

- **主光（Key Light）**：侧面45度
- **补光（Fill Light）**：对侧，弱
- **轮廓光（Rim Light）**：背后，冷色

### 36. 武侠光影核心

- 侧光剑气
- 冷暖对决
- 高对比度

### 37. 古风光影核心

- 逆光剪影
- 顶光光束
- 留白构图

### 38. 赛博光影核心

- 霓虹RGB
- 运动模糊
- 俯仰倾斜

### 39. 情感光影核心

- 柔光
- 逆光
- 暖色调

### 40. 悬疑光影核心

- 顶光
- 低对比
- 单一光源

### 41. 武侠剑光制造

```
side lighting, blade reflecting cold gleam,
steel sword with white highlight,
dramatic shadow on face
```

### 42. 雾气氛围营造

```
mist, fog, volumetric light,
god rays, dust particles in light beam
```

### 43. 光影一致性原则

同一部剧里，灯光风格要统一

### 44. 夜景灯光处理

- 月光（冷蓝）
- 霓虹（彩色）
- 单一光源（手电/蜡烛）

### 45. 室内灯光处理

- 窗光（自然）
- 人工光（台灯/吊灯）
- 混合光（冷暖对比）

## 四、构图设计（12个知识点）

### 46. 构图18公式

详见《AI漫剧创作方法论》文章：三分法/对角线/框架/留白/纵深/对称/三角/斐波那契/引导线/封闭式/开放式/荷兰角/中心/隧道/并列/压迫/打破第四面墙/景别对比

### 47. 三分法

- 横向/纵向各三等分
- 主体放在交叉点
- 最通用的构图

### 48. 框架构图

- 用门窗/树枝等框架
- 聚焦主体
- 增加层次

### 49. 留白构图

- 中国画精髓
- 制造意境
- 给想象空间

### 50. 对角线构图

- 制造动感
- 适合动作场景
- 引导视线

### 51. 构图与情绪

| 构图 | 情绪效果 |
|------|---------|
| 三分 | 稳定 |
| 中心 | 聚焦 |
| 框架 | 封闭 |
| 留白 | 意境 |
| 对角线 | 动感 |

### 52. 景别与构图配合

| 景别 | 推荐构图 |
|------|---------|
| 远景 | 三分法、中心 |
| 全景 | 三分法、对角线 |
| 中景 | 三分法、框架 |
| 近景 | 中心、留白 |
| 特写 | 中心、紧凑 |

### 53. 竖屏构图特点

- 人物居中或偏上
- 上下留白少
- 适合近景/特写

### 54. 横屏构图特点

- 三分法更常用
- 视野宽广
- 适合建立镜头

### 55. 荷兰角用法

- 失衡感
- 紧张/醉酒/战争
- 心理状态外化

### 56. 封闭式 vs 开放式

- 封闭式：完整、稳定
- 开放式：不完整、焦虑

### 57. 构图检查

- 主体是否突出？
- 引导线是否清晰？
- 留白是否合适？

## 六、剪辑与后期（12个知识点）

### 70. 剪辑节奏类型

恒定/加速/减速/跳跃/静止

### 71. 镜头时长参考

| 景别 | 时长 |
|------|------|
| 大远景 | 5-8秒 |
| 远景 | 4-6秒 |
| 全景 | 3-5秒 |
| 中景 | 2-4秒 |
| 近景 | 1.5-3秒 |
| 特写 | 1-2秒 |
| 大特写 | 0.5-1秒 |

### 72. 音乐卡点

| 音乐节点 | 剪辑动作 |
|---------|---------|
| 前奏 | 建立镜头 |
| 主歌 | 切入关键 |
| 副歌 | 快切或定格 |
| 间奏 | 场景转换 |

### 73. 10种转场技巧

叠化/闪切/黑场/动作匹配/声音先入/POV/升降格/遮挡切换/色彩渐变/记忆闪回

### 74. DaVinci调色流程

```
Raw → 还原 → 一级校色 → 二级校色 → 风格化 → 交付
```

### 75. 调色节点结构

串行节点/并行节点/键/遮罩节点

### 76. HDR注意事项

- 亮度范围更大
- 色域更宽
- 不要过度饱和

### 77. 交付格式

| 平台 | 格式 | 分辨率 | 码率 |
|------|------|--------|------|
| 抖音 | H.264 | 1080x1920 | 8-15Mbps |
| B站 | H.264 | 1920x1080 | 6-20Mbps |
| YouTube | H.265 | 4K/1080p | 20-50Mbps |

### 78. 字幕规范

- 字体清晰
- 大小合适
- 位置不遮挡主体

### 79. 音效配合

- 动效音增加真实感
- 环境音建立氛围
- 关键音强化节奏

### 80. 快门速度与景深

- 高速快门：凝固瞬间
- 低速快门：运动模糊

### 81. 剪辑自检

- [ ] 时长是否符合？
- [ ] 节奏是否流畅？
- [ ] 景别是否有变化？
- [ ] 音乐是否卡点？
- [ ] 高潮是否"卡一半"？
