## My TV 是什么？

My TV 是一款开源的安卓电视直播播放器，主要特点：

- 完全免费，无任何内购
- 无广告，看完整场球赛也不会弹出广告
- 开源透明，不会有恶意扣费
- 支持自定义直播源，可以自己配置频道
- 支持 Android 5.0 以上所有设备（TV / 手机 / 平板）

## 快速开始

### 1. 安装 APK

下载 APK 文件后，通过 U 盘插入电视安装，或者用「adb install」推送到设备。

如果是第一次安装，需要在电视设置里开启「允许安装未知来源应用」。

### 2. 打开 APP

第一次打开默认会加载官方频道列表。

界面通常包含：

- 电视频道列表（央视、卫视、地方台）
- 分类切换（CCTV、卫视、地方、体娱、影视等）
- 设置入口

### 3. 开始观看

选频道 → 播放 → 换台，就是这么简单。

## 自定义直播源（进阶玩法）

My TV 的精髓在于**支持自定义直播源 JSON**。

### 什么是直播源？

直播源本质上是一个 URL，指向某个视频流地址（M3U8 / FLV / HLS 等格式）。好的直播源稳定、快速、频道全。

### 导入自定义源

在设置里找到「直播源管理」或「添加源」：

```json
{
  "name": "我的源",
  "url": "https://example.com/tv/live.txt"
}
```

### 推荐直播源

以下是一些常用的免费直播源（使用前请自行测试稳定性）：

**源1（综合）：**
```
https://example.com/tv/live.txt
```

**源2（地方台为主）：**
```
https://example.com/local/live.txt
```

> 直播源经常失效，建议多收藏几个，发现挂了及时更换。

### 如何找直播源？

几个思路：

- GitHub 搜索 my-tv 或 tv-source，会有人维护更新
- 酷安、贴吧等社区经常有人分享
- 一些直播源聚合网站会定期更新

## 常见问题

**Q：播放卡顿怎么办？**
- 切换其他源试试
- 检查网络环境，可能是宽带质量问题
- 部分频道有多个线路，轮流尝试

**Q：频道全黑或提示加载失败？**
- 该源已失效，需要更换新源
- 检查网络是否正常
- 尝试重启 APP

**Q：如何在手机上使用？**
- 和 TV 一样，下载 APK 安装即可
- 手机界面会自动适配，操作逻辑相同

**Q：支持回看/点播吗？**
- 不支持，My TV 是纯粹的直播播放器
- 如果需要回看，可以配合其他 APP 使用

**Q：安全吗？**
- 开源项目，代码透明
- 建议只从可信渠道下载 APK

## 进阶：搭建自己的直播源服务器

如果你有一定技术能力，可以自己搭建一个直播源管理服务器：

1. 找一台常年在线的 VPS（带宽越大越好）
2. 部署一个直播源聚合程序
3. 将整理好的 M3U8 链接聚合为一个 JSON 文件
4. 将 JSON 文件 URL 填入 My TV

这样你就有了一个**完全自己掌控**的直播源服务，完全不用担心第三方跑路。
