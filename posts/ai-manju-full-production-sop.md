# AI漫剧制作全流程SOP：从零到S级的完整作战手册

> 本文是我经过大量实战总结出的AI漫剧制作完整流程，适合想要系统化提升的创作者。

---

## 一、我的制作失败经历

在总结这套SOP之前，我踩过很多坑：

| 阶段 | 我犯过的错 |
|------|-----------|
| 剧本 | 直接开始写分镜，没有先分析风格 |
| 风格 | 做到一半发现光影不统一 |
| 角色 | 每集换一张脸，观众完全认不出 |
| 运镜 | 全是特写，看着像PPT |
| 节奏 | 高潮处没有切断，提前展示了结果 |
| 输出 | 没有统一格式，后期剪辑混乱 |

**这些错误让我浪费了大量时间，也做出了很多B级甚至C级的内容。**

后来我总结了这套SOP，按照流程走，几乎不会再犯同样的错误。

---

## 二、AI漫剧制作6步法

### Step 1：项目立项（10分钟）

**做什么：**
- 明确项目类型（武侠/情感/悬疑/古风/赛博）
- 确定时长目标（30秒/60秒/90秒/120秒）
- 确定发布平台（抖音/快手/B站/YouTube）
- 确认核心Hook（开头3秒抓什么）

**输出模板：**
```
【项目名称】
【类型】
【时长】
【平台】（影响竖屏/横屏）
【核心Hook】
```

**平台适配注意：**

| 平台 | 竖屏/横屏 | 时长 | 风格 |
|------|-----------|------|------|
| 抖音/快手 | 竖屏 | 30-60秒 | 快节奏 |
| B站 | 横屏/竖屏 | 60-120秒 | 可慢节奏 |
| YouTube | 横屏 | 60秒+ | 专业感 |

---

### Step 2：剧本创作（30-60分钟）

**做什么：**
- 写出完整的剧本（不是大纲）
- 标注每个场景的四元素：景别/人物/动作/情绪
- 设计Hook和结尾截断点
- 检查节奏是否够紧凑

**四元素标注示例：**
```
【场景1】竹林初遇
时间：黄昏
地点：外景-竹林

[远景] [环境] [建立氛围] [神秘]
薄雾缭绕，夕阳透过竹叶洒落

[中景] [男剑客] [缓步走来] [警觉]
黑衣剑客手持长剑，目光警惕

[近景] [女剑客] [回眸] [惊讶]
白衣女子回首，四目相对
```

**剧本自检问题：**
- [ ] 开头3秒有没有钩子？
- [ ] 是否有明确的情绪曲线？
- [ ] 结尾是否截断在期待点？
- [ ] 时长是否合理？（每分钟约100字剧本）

---

### Step 3：风格确定（10分钟）

**做什么：**
- 根据剧本类型确定主风格
- 选择对应的运镜/光影/构图/道具方案
- **重要：禁止混用不同风格的元素**

**风格选择速查：**

| 剧本类型 | 运镜 | 光影 | 构图 |
|---------|------|------|------|
| 武侠/动作 | 动态+快速 | 侧光剑气 | 纵深三分 |
| 情感/爱情 | 缓慢推进 | 柔光逆光 | 框架亲密 |
| 悬疑/惊悚 | 固定+慢移 | 顶光低对比 | 低角压迫 |
| 古风/仙侠 | 诗意摇移 | 逆光留白 | 对称写意 |
| 赛博/科技 | 俯仰倾斜 | 霓虹RGB | 荷兰角 |
| 宣传片 | 稳定跟拍 | 写实自然 | 三分构图 |

**教训：确定风格后，禁止中途换风格！**

---

### Step 4：角色设计（20-40分钟）

**做什么：**
- 设计角色的外貌描述（≥200字）
- 生成角色的三视图（正面/侧面/背面）
- 选定后锁定，后续全部以此为参考

**三视图生成提示词：**

**正面：**
```
[详细脸型+眉+眼+鼻+唇+肤色+发型描述]，
正面照，正面光源，清晰五官，
无角度偏斜，作为角色基准图
```

**侧面：**
```
[同上脸型描述]，
标准90度侧脸，侧面向右/左，
侧脸轮廓：颧骨/下颌线/鼻梁，
侧向光源，轮廓光强化
```

**背面：**
```
[发型背面+肩线+后颈描述]，
背面全身照，镜头正后方，
背光光源，轮廓逆光，
服装背面细节清晰
```

**教训：一定要做三视图！三视图！三视图！**

---

### Step 5：分镜脚本制作（60-120分钟）

**做什么：**
- 根据剧本分配镜头
- 写每个镜头的完整提示词
- 设计运镜/灯光/构图/音效
- 检查景别变化和节奏

**分镜脚本标准格式：**

| 序号 | 时长 | 场景 | 景别 | 提示词 | 运镜 | 灯光 | 构图 | 音效 |
|------|------|------|------|---------|------|------|------|------|
| 1 | 5秒 | 竹林 | 远景 | XXXX | 横移 | 自然 | 三分 | 风 |

**镜头分配公式：**
```
开场(15%)：全景/中景建立 → 2-3个镜头
发展(50%)：中景/近景交替 → 节奏推进
高潮(25%)：特写/大特写 → 情绪爆发
结尾(10%)：拉远/留白 → 戛然而止
```

**教训：提示词要写完整，不能偷懒！**

---

### Step 6：生成与剪辑（60-180分钟）

**做什么：**
- 按分镜脚本生成所有镜头
- 剪映剪辑合成
- 配乐/音效/字幕
- 输出最终成品

**生成检查清单：**
- [ ] 每个镜头是否符合分镜要求？
- [ ] 角色是否与设定一致？
- [ ] 光影风格是否统一？
- [ ] 运镜是否流畅？

**剪辑自检清单：**
- [ ] 时长是否符合目标？
- [ ] 节奏是否流畅？
- [ ] 景别是否有变化？
- [ ] 音乐是否卡点？
- [ ] 高潮是否"卡一半"？

## 三、实战时间分配

| 阶段 | 时间 | 说明 |
|------|------|------|
| 项目立项 | 10分钟 | 不能省的规划 |
| 剧本创作 | 30-60分钟 | 根据时长调整 |
| 风格确定 | 10分钟 | 选定就不改 |
| 角色设计 | 20-40分钟 | 三视图一定要做 |
| 分镜制作 | 60-120分钟 | 最重要的环节 |
| 生成+剪辑 | 60-180分钟 | 根据经验和工具 |

**一个60秒的漫剧，平均需要4-6小时完成。**

## 四、常见问题与解决

**Q1：总是控制不好时长怎么办？**
解决：用计时器实时监控。每个镜头都算好时长的，超时就砍。

**Q2：角色总是换脸怎么办？**
解决：一定做三视图。生成时上传三视图作为参考。

**Q3：光影总是不统一怎么办？**
解决：在分镜阶段就写清楚每个镜头的灯光方案，生成时不要随意改。

**Q4：节奏总是太拖沓怎么办？**
解决：记住"卡一半"原则。高潮处直接切断，不展示完整结果。

**Q5：运镜总是太单调怎么办？**
解决：检查分镜脚本，确保有景别变化。固定镜头不超过连续3个。

## 五、进阶经验

### 经验1：建立自己的素材库

每次做完一个项目，我会把：
- 好的提示词模板 → 存入笔记
- 好的角色设计 → 截图保存
- 好的灯光方案 → 截图保存
- 踩过的坑 → 记录教训

**3个月后，做新项目的速度提升了一倍。**

### 经验2：遇到问题定向拉片

当我不知道某个场景该怎么拍时，我会去找同类型的电影/剧/广告，拉片学习。

不是抄，而是分析：
- 这个镜头为什么是这个景别？
- 光源在哪里？
- 运镜是怎么做的？

### 经验3：保持学习，但不要信息过载

AI工具发展很快，但核心的视听语言基础几十年没变。

**策略：工具更新就学，但基础功（剧本/运镜/灯光/剪辑）要持续打磨。**

## 结语

记住核心：
1. **规划比执行重要** — 不要急着开始，先想清楚
2. **风格一旦确定就不要换** — 混用是最大的坑
3. **三视图是角色一致性的生命线**
4. **提示词要写完整，不能偷懒**
5. **卡一半是高点击的秘诀**

祝你做出S级作品！
